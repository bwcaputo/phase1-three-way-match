Three-Way Match Agent — System Architecture

What lives where, and how one invoice check moves through the stack

Your Laptop

Python process

The Agent

Lives in src/ and scripts/. Runs only while the command is active. Talks to Claude over the public internet and to Odoo over localhost:8069.

Docker Desktop

Container

Odoo 17

Open-source ERP. Serves the web UI on port 8069 and an XML-RPC API the agent calls. Stores nothing itself — everything goes through Postgres.

Container

Postgres 15

The database under Odoo. Holds vendors, products, purchase orders, receipts, and invoices. Invisible to the agent — only Odoo talks to it.

Cloud

API call

Anthropic Messages API

Where Claude lives. The agent sends the conversation so far, Claude decides which tool to call next, Python executes it against Odoo, and the loop continues until Claude writes a final summary.

What Claude never does

Claude never opens a database connection, never computes a dollar amount, never decides the final action. Math and matching live in src/match.py on your laptop.

One invoice check, step by step

  1. You run the command python -m scripts.run_agent BILL/2026/04/0001 starts a Python process on your laptop.
  2. Agent calls Claude Python sends the system prompt, the invoice number, and the five tool schemas to Anthropic.
  3. Claude picks a tool First call: fetch_vendor_invoice. Python runs that against Odoo via XML-RPC.
  4. Odoo queries Postgres Inside the Docker network, Odoo translates the request into SQL, hits Postgres, returns JSON.
  5. Python hands result to Claude Claude now sees the invoice data and picks the next tool: fetch_purchase_order.
  6. Loop repeats Claude calls fetch_goods_receipt, then check_for_duplicate_invoices.
  7. Pure-Python matcher runs Claude calls run_three_way_match. Python does the comparison, not the model. Returns action, discrepancies, audit trail.
  8. Claude narrates Final turn: Claude writes the English summary citing numbers the matcher produced.
  9. Terminal prints the result Rich tables, approve/route/block badge, audit trail. You never paid anything. A human still does that.
